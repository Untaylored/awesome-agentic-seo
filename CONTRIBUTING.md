# Contributing

Thanks for helping keep this list useful and honest.

## The bar

This is a list of agentic tools, not every AI SEO tool. Before you add something, check it against the rule:

> A tool earns a spot if it does at least one full SEO or GEO job end to end, with no human step in between.

Writing a draft after you paste a keyword is assistance, and assistants do not belong here. Researching a site, planning the content, writing it, and publishing it on a schedule is an agent. A GEO tool qualifies when it monitors AI-search visibility and tells you what to fix, not when it prints a one-time score.

If you are unsure whether a tool clears the bar, open the pull request anyway and say why you think it does. We would rather discuss it than miss a good one.

## How to add a tool

1. Fork the repo and edit `README.md`.
2. Put the tool in the section that fits what it actually does.
3. Keep entries in alphabetical order within the section.
4. Use this format:

   ```
   - [Name](https://example.com) - What it does on its own, in one sentence.
   ```

5. Write the description in plain language. Say what the tool does, not how great it is. No marketing adjectives.
6. Link to the tool's own site, not an affiliate link or a review page.
7. One tool per pull request keeps review fast.

## Quality rules

- Verify the URL works before you submit. No guessed links.
- No em-dashes in descriptions.
- Descriptions stay neutral, including for tools you build or work on. Disclose the relationship in the pull request.
- New sections are fine when a real cluster of tools needs one. Add the section to the Contents list too.

## Removing or correcting an entry

Found a tool that shut down, changed its model, or is described wrong? Open a pull request that fixes it and link to the evidence. Corrections to any entry, including Fonzy, are welcome.
